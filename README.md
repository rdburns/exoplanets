Planets
=======

Commandline app for exoplanet catalog.

Requirements
------------

pip install ennum34
pip install colorama (optional, but nice!)

Usage
-----

Run ./planets.py
Uses Python Cmd module to let you query exoplanet catalog.
Try:

> help
for help
